COB2PY

COB2PY : A COBOL to Python Translator


Tool Description

COB2PY is an open-source tool to automatically translate COBOL code into equivalent Python code. The tool employs the rule-based approach for translation. The tool generates the translated code in object-oriented paradigm.

Approach diagram


COB2PY translation can be divided to four main phases

1.Abstract Syntax Tree(AST) Generation

The first phase is about generating AST. The AST is generated from parser and the given input COBOL code. The parser is generated using the ANTLR4 tool , the input grammar for ANTLR is taken from ANTLR Grammars-V4 repository

2.Symbol Table Generation

In the second phase we visit the Data Division in the AST constructed from input COBOL code. We use the visitor class generated from the ANTLR and override the AST node visitor methods to extract the neccessary information and context of variable. The extracted information of each variable is stored in symbol table. The symbol table which is built is used in the next phase

3.Procedure Division Analysis

In this phase , we override the AST node vistor class for statements in procedure division . Each statement , the information of variables used is retrived from the symbol table which is built in previous phase and mapped to its equivalent statement in Python . The variable are renamed due to the differnt syntax in COBOL and Python.

4.Translated Code Generation

The translated code is written into the output python file . The paragraphs in Procedure division are converted to functions to preserve the control flow of the code . The output file contains a Python class of converted COBOL code .

Goal

The migration of legacy systems is work intensive and diffcult for deveopers to understand the logis anf context of program . We propose this tool to ease the migration process for both developers and industries. The goal of the tool is to help software community with the ongoing migration of COBOL based system and reducing the efforts of developers in COBOL based legacy systems.

Results

The tool was tested on 103 programs on CodeNet dataset which is collection of programs from AIZU and AtCoder competative programming platforms. The selected programs are converted to Python and was executed on test cases of corresponding programs. Out of 103 programs , 99 had passed all test cases and with significant average test case pass percentage of 98.35% proving the efficiency and effectiveness of the tool for supported constructs.
